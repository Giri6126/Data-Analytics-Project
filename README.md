# 🌍 Global Energy Consumption Visualization Dashboard

## 📌 Overview

The **Global Energy Consumption Visualization Dashboard** is a data analytics project that focuses on analyzing and visualizing global energy usage trends. The project transforms complex datasets into interactive and easy-to-understand visual insights, enabling users to explore energy consumption patterns across countries and time periods.

This project demonstrates the power of **data analytics, preprocessing, and visualization techniques** in solving real-world problems.

---

## 🎯 Objectives

* Analyze global energy consumption data
* Identify trends and patterns across countries and years
* Compare renewable and non-renewable energy usage
* Develop interactive visualizations and dashboards
* Enable users to explore data dynamically

---

## 📂 Dataset

The dataset used in this project is sourced from:

* Our World in Data (OWID Energy Dataset)

### Dataset Features:

* Country-wise energy consumption
* Year-wise historical data
* Renewable and non-renewable energy data
* Additional indicators like GDP and population

## ⚙️ Project Workflow

1. **Data Collection**

   * Download dataset from OWID

2. **Data Preprocessing**

   * Handle missing values
   * Clean and filter relevant data

3. **Exploratory Data Analysis (EDA)**

   * Analyze trends and patterns
   * Compare energy consumption

4. **Data Visualization**

   * Line charts (trend analysis)
   * Bar charts (country comparison)
   * Pie charts (energy distribution)

5. **Dashboard Development**

   * Interactive UI
   * Filters (country, year, energy type)

## ▶️ How to Run the Project

### 🔧 Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/global-energy-dashboard.git
cd global-energy-dashboard
```

### 📦 Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

### ▶️ Step 3: Run the Application

```bash
python app.py
```

### 🌐 Step 4: Open in Browser

```
http://127.0.0.1:5000/
```
